对外接口

通过模块对外的methods进行函数调用

接口KEY 类型 接口说明
getActive () => { name: string;index: number; } 返回当前处于激活的tab
next () => string 切换到下一个tab，返回激活的tab的name
sub () => string 切换到上一个tab，返回激活的tab的name
getAllChild (filter?: (item: TabItem) => boolean) => string[]; 返回全部tab， filter，如需要过滤可传入过滤函数，传递给Array.filter
changeActive (name: string) => void; 切换tab｜
changeDisabled (value: string, disabled: boolean) => void;｜ 切换指定的tab的disable的状态｜
changeAllDisabled (disabled: boolean) => void;｜ 切换所有的tab的disable的状态｜
