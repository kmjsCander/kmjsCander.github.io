表格cell类型

text

文本(默认该类型）

image

展示图片（目前仅支持展示一张）

特殊配置泛指params选项的值：

mapText

数据与汉字的映射 一般用于后台返回状态的code，但是要展示状态的文字

特殊配置泛指params选项的值：

rangeText

一般用来显示时间范围 xxxx～yyyy

特殊配置泛指params选项的值：

listView

展示列表数据

特殊配置泛指params选项的值：

fileListView

展示列表文件， 并支持点击文件名现在

表格动态入参

通过useTable传入

参数KEY 类型 参数说明
reqBody { [l: string]: any } 该对象内的数据，会作为请求表格数据的query
pathBody string[] 该数组内的字符串会默认拼接到请求表格数据的url上面
dataFormatter (tableData: any[]) => any[] 该函数，会在数据请求完成后调用。可以对请求数据做进一步的处理
beforeRequest (requestObj: { url: string; params: any; }) => Promise<{ url: string; params: any } | null> 该函数在每次请求前调用根据返回的值进行数据请求,如果返回null则表示取消当次请求

表格对外接口

通过useTable返回的methods进行函数调用

接口KEY 类型 接口说明
refresh () => void 触发从新搜索，当前页面搜索
refreshAll () => void // 从第一页搜索
getSearchData () => Promise<{ [l: string]: any }>; 返回当前表格查询数据的入参，在调用beforeRequest后返回 有可能会返回null
getTableData ()=> any[] 获取当前表格渲染的数据
getCheckData ()=> any[] 获取当前表格多选框钩上的数据
setSearchOptions (key: string, data: {label:string; value:string}[]) => void 设置搜索中某些搜索的项可被用户选择的值，比如说某个select的数据来自于后台数据, 通过key去设置

表格对外回调

回调KEY 类型 接口说明
searchSearchChange {key:string; data: any} 高级搜索中某个控件值发生变化时触发
